# Cheat Sheet For Starting New Express.js Backend App

## Initialized 
```
1. npm init
```

## Install required packages one by one

```
1. npm i express
2. npm i mongoose
3. npm i cors
4. npm i dotenv
5. npm i bcrypt 
6. npm i jsonwebtoken
```

## Install required packages in one line
```
npm i express mongoose cors dotenv bcrypt jsonwebtoken
```
